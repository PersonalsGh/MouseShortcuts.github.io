About MouseShortcuts

What is MouseShortcuts?

MouseShortcuts is a lightweight Windows desktop application that maps common keyboard shortcuts to intuitive mouse gestures. Instead of reaching for Ctrl+C, Ctrl+V, or PrintScreen, you perform simple mouse actions — double-clicks, combination clicks, swipes, and edge hovers — to trigger the same commands instantly.

The app runs quietly in the background and intercepts mouse input system-wide. When it detects a recognized gesture pattern, it executes the corresponding action. When it doesn't detect a gesture, your mouse behaves completely normally. There's no learning curve to worry about — the gestures are designed to feel natural from the first use.

Why We Built It

The idea for MouseShortcuts came from a simple frustration: the constant back-and-forth between mouse and keyboard during everyday tasks. Selecting text with the mouse, reaching for the keyboard to copy, going back to the mouse to navigate, reaching for the keyboard again to paste — it's a cycle that repeats hundreds of times a day.

We realized that the mouse is already in your hand for most of these operations. Why not let it handle the shortcuts too? The result is an app that keeps your hand where it already is, saving time and reducing the repetitive strain that comes with constant device switching.

The Developer

MouseShortcuts is an indie-developed project — built, maintained, and supported by a solo developer passionate about creating tools that make daily computing more efficient. Every feature, every gesture, and every line of code is crafted with care and a focus on the user experience.

The app is built with modern technologies: Electron for the desktop shell, React for the interface, and AutoHotkey for the low-level gesture engine. This combination provides a polished UI with the system-level access needed for reliable gesture detection.

Transparency

We believe in transparency. MouseShortcuts does not collect usage data, telemetry, or any information about what you do on your computer. The only personal data we store is your email address for account authentication — and you can delete that at any time from within the app.

The desktop application is distributed via GitHub Releases, so you can always verify exactly what you're downloading. If you have questions about how the app works or what it does, feel free to reach out.

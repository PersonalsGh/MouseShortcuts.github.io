2. Description of Service

MouseShortcuts is a free, ad-supported desktop application that enables users to map eight intuitive keyboard shortcuts to mouse gestures. The Software runs a helper executable in the background to listen for specific mouse events and translate them into keyboard actions.

4. Background Process Transparency

To provide its core functionality, the Software runs a helper executable (powered by AutoHotkey) as a background process. This helper:

  • Listens for specific mouse gestures and translates them into keyboard shortcut actions.
  • Operates at the operating-system input-hook level to intercept mouse events.
  • Is started and stopped through the application's user interface.
  • Is NOT malicious software, spyware, or a keylogger — it only processes the predefined mouse gestures documented in the application.
  • May be flagged by some antivirus software as a "potentially unwanted program" due to its use of input hooks; this is a false positive.
